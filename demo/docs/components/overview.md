# Components

All styled Docusaurus components and utility classes — aligned with the Rackscope design system.

## What's covered

| Component | Description |
| --- | --- |
| [Typography](./typography) | Headings, blockquote, `kbd`, `hr`, inline formatting |
| [Banners](./banners) | Note, tip, info, warning, danger — Toast style |
| [Code Blocks](./code-blocks) | Syntax highlighting, titles, line highlighting |
| [Tables & Lists](./tables-lists) | Tables, ordered/unordered lists, images |
| [Tabs](./tabs) | Underline tabs, with code blocks, with text content |
| [Navigation](./navigation) | Breadcrumb, pagination |
| [Badges & Labels](./badges-labels) | Status labels, HTTP method badges, release tags |
| [Buttons](./buttons) | `.btn-*` utility classes for MDX pages |
| [Utilities](./utilities) | `.steps`, `.timeline`, `.list-check` |
